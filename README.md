## Hotsite Sistema Logosófico de Educação

Projeto de criação de um hotsite para o Sistema Logosófico de Educação com fins acadêmicos para o Projeto Aplicado da Faculdade XP Educação.

No momento, o projeto encontra-se em fase de desenvolvimento.
